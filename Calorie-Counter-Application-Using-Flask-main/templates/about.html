About Us

      In today's fast-paced world, where unhealthy food options are abundant and sedentary lifestyles are prevalent, maintaining a healthy weight and overall well-being can be challenging. This Calorie Counter Application, developed using Python and Flask, is a user-friendly web application that helps users keep track of their daily caloric intake and maintain a healthy lifestyle.

   This applicationâ€™s backend is developed using Python, and the frontend is developed using HTML, CSS, Bootstrap, and JavaScript to create a responsive and interactive user experience. By using the Flask framework, the application provides an interactive and user-friendly interface where users can register, log in, and input their consumed food items and quantities, enabling the application to calculate and display their total calorie consumption throughout the day.

    The application utilises the Oracle MySQL database to store user data, including meal entries and corresponding calorie information. It empowers users to take control of their health by effortlessly monitoring and managing their calorie intake, leading to improved well-being and healthier lifestyles.
